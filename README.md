# AEGIS Governance Platform

A responsive static prototype for higher-education AI transparency, process journaling, institutional governance, and compliance.

## Local preview

Open `index.html` directly, or run a local server:

```bash
python -m http.server 3000
```

Then visit `http://localhost:3000`.

## Deploy to Vercel

1. Create a GitHub repository and push this folder.
2. In Vercel, select **Add New → Project** and import the repository.
3. Keep **Framework Preset** as `Other`.
4. Leave Build Command and Output Directory empty.
5. Click **Deploy**.

No environment variables or build step are required.
